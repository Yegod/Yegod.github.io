使用 Hexo & Github建一个博客
本人只在mac上操作过，如果在Windows上面可能略微有点出入，如果遇到某些命令不能执行，请自行查阅替代命令。

什么是Hexo？
hexo官方文档

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

什么是Github？
Github 维基百科

使用二者搭建博客的好处
可以不用购买域名和服务器，配置比较简单，在搭建的过程中还可以学习使用github！

步骤
新建第一篇博客
1.首先，我们先需要安装git和node.js，并且要注册Github

node.js
git
github
2.接下来我们需要安装Hexo,打开终端(Terminal)

1
npm install -g hexo-cli
3.进入一个安全目录

1
cd ~/Desktop
4.新建一个文件夹并且进入

1
2
mkdir Blog
cd Blog
5.依次执行

1
2
3
hexo init myBlog
cd mylog
npm i
6.新建第一篇博客

1
hexo new blog-1.md
7.编辑上面这个md文件，这个文件的内容就是你的博客内容

1
open ~/Desktop/Blog/myBlog/source/_posts/blog-1.md
8.进入Github,点击绿色按钮【new repository】，新建一个仓库的名称为【你的用户名.github.io】的仓库，比如我的是【mysteryven.github.io】输入名称之后什么都不要点，直接点击下面的【create repository】

打开命令行
1
open _config.yml
第6行title改为你想的标题
第9行author改为你的名字
最后一行type改为

1
type: git
type下面新加一行，左边和type对齐(type:和repo:后面都有空格)

1
repo: git@github.com:mysteryven/mysteryven.github.io.git
上面的地址是你们github上面的ssh的地址，如果找不到可以修改我用户名的部分转化成你们的。

10.依次执行

1
2
npm install hexo-deployer-git –save
hexo deploy
11.此时您就可以预览了第一篇博客了，预览地址打开【你的用户名.github.io】对应的仓库的setting，往下面翻就能找到

第二篇博客
1.依次执行

1
2
3
4
hexo new 第二篇博客
open new ~/Desktop/Blog/myBlog/source/_posts/第二篇博客.md
hexo generate
hexo deploy
上面open后面的路径是您执行上一步操作命令行给出的那个

完成啦~ 以后还想继续添加的话就按照这个步骤！
给博客换主题
1.从主题合集里面选择一个自己满意的主题
2.进入他的github首页
3.复制它的ssh地址,下面假定我选择了一个主题
4.

1
cd themes
5.

1
git clone git@github.com:iissnan/hexo-theme-next.git
6.

1
cd ..
7.
将 _config.yml 的第 75 行改为

hexo-theme-next，

上传源代码
这一步的目的是为了备份您博客里面的数据，如果有一天您丢掉了【myBlog】文件夹里面的内容，那么还可以通过这次的备份还原

在Github创建名为【blog-generator】的空仓库
在接下来的页面中点击截图中的ssh，然后按照【create a new repository on the command line】的步骤执行
这样之后就可以啦，每次写完代码，就用git的一系列操作add/commit/push等传到这个仓库里面就可以了
完成
